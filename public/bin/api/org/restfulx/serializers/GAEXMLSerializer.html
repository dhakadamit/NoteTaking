Packageorg.restfulx.serializers
Classpublic class GAEXMLSerializer
InheritanceGAEXMLSerializer XMLSerializer GenericSerializer

Serialises RxModel instances to XML and back. XML produced is quite different from the form used by SimpleXMLEncoder/Decoder. Target format is expected to be more GAEDS-like, with a lot less formalism and a lot more conventions.
